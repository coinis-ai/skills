# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this skills bundle, please report it
privately by emailing **support@coinis.com**. Include a description of the
issue, the affected files or skills, and steps to reproduce if possible.

Please do not open a public issue for security reports. We aim to acknowledge
reports within a few business days, and we ask that you allow a reasonable
disclosure window before publishing any details.

## Supported Versions

The latest released version on the `main` branch is supported. Security fixes
are applied to that line; older tags are not maintained.
